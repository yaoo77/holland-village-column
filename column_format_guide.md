# 河村真紀子様コラムの正しいフォーマット

## 基本構造

```
【今日の金融トピック】
[ニュースタイトル]
〜[情報源]
↑ [導入の一言（ニュースに応じて変わる）]
[本文...]

[締めの言葉（オプション）]
#HVMC_Morningcolumn
```

## 固定要素

1. **【今日の金融トピック】** - 必須、冒頭に配置
2. **ニュースタイトル** - 情報源から引用または要約
3. **〜[情報源]** - Bloomberg、Nikkei、WSJ等
4. **↑** - 矢印記号、導入部の前に配置
5. **#HVMC_Morningcolumn** - 必須、最後に配置

## 可変要素

### 導入部（↑の後）
- 「なんやなんや？」
- 「[具体的な数値や事実]」
- 「『[引用やフレーズ]』と言われるか？」
- など、ニュースに応じて自由に変わる

### 本文
- 段落構成は自由
- 改行を適切に入れる（読みやすさ重視）
- 700文字前後を目標

### 締めの言葉（オプション）
- 「良い一日を！」
- 「⚠️投資は自己判断で！」
- など、コラムの内容に応じて選択

## 改行のルール

サンプルコラムから観察される改行パターン：

1. **タイトルと情報源の後**: 必ず改行
2. **導入部の後**: 改行して本文へ
3. **段落の区切り**: 意味のまとまりごとに改行
4. **強調したい一文**: 前後に改行を入れる
   - 例：「でもね。」「いやー、なんかさ。」
5. **締めの前**: 改行を入れる

## 文字数の目安

- **最短**: 443文字（コンパクトなコラム）
- **平均**: 700文字前後
- **最長**: 887文字（詳細な分析）

### 例1: シンプルなコラム

```

よく私が「これからは中国🇨🇳株」と言うと「え、でもデフレなんでしょ？」「景気悪いって聞いたよ」なんて心配のお声が。
多分ね、こういうことなんです。
中国だけじゃなく、世界中が「⚪︎⚪︎の国は景気が良い」とザックリ言えなくなってる。
何故なら国内でも二極化してるから。
富裕層は好景気、貧困層はデフレ。
そんな「ドメスティック二極化」がどこの国でも起きてるから「あれ？景気いいの？悪いの？どっち？？」という感想になる。
ちなみに二極化が進むと、株価は富裕層の方に引っ張られる。何故なら、株に投資できる人の多くは貧困層ではないから。
中国株、引き続きウォッチしてます👀
#HVMC_Morningcolumn
```

## 重要な注意点

1. **改行は必須**: 長い文章を詰め込まない
2. **文字数は700文字前後**: 短すぎても長すぎてもダメ
3. **タイトルと情報源は必須**: 権威性を示す
4. **導入部は固定ではない**: ニュースに応じて変える
5. **締めの言葉は柔軟**: コラムの内容に応じて選択

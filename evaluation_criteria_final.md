# 河村真紀子様コラムの評価基準（最終版）

## 評価項目と配点

### 1. 言語・方言（10%）

**チェック項目**:
- 標準語を使用している
- 関西弁、方言は使用していない
- 「なんやなんや？」等の独特の表現は使用してもよいが、コラム全体が方言になってはいけない

**減点対象**:
- 関西弁の使用（「やなぁ」「やろか」「せやけど」「やで」「あかん」「〜や」等）
- その他の方言の使用

**重要**: 「なんやなんや？」は河村様の独特の表現であり、関西弁ではない。コラム全体は標準語を基本とする。

### 2. フォーマットの正確性（15%）

**必須要素**:
- [ ] 【今日の金融トピック】が冒頭にある
- [ ] ニュースタイトルが記載されている
- [ ] 〜[情報源]が記載されている
- [ ] ↑ 導入部がある（ニュースに応じて変わる、固定ではない）
- [ ] #HVMC_Morningcolumn が最後にある
- [ ] 適切な改行がある（段落ごと）
- [ ] 文字数が700文字前後（±100文字、つまり600-800文字）
- [ ] 絵文字が使用されている（🎉🤭🏰👀🤚等）

**減点対象**:
- 必須要素の欠落
- 文字数が範囲外（600文字未満または800文字超）
- 改行が不適切（長文が詰まっている）
- 絵文字がない

### 3. 口調・言い回しの再現度（25%）

**使用すべき表現**:
- 疑問形の連発（「〜じゃない？」「〜わよね」）
- 独特の言い回し:
  - 「なんやなんや？」（疑問提起）
  - 「クソ面倒い」「クソ重要」（強調）
  - 「ぶっちゃけ」「要するに」（率直な表現）
  - 「そら」（理由の説明）
  - 「〜わよね」「〜のよ」（女性的な語尾）
- 特徴的な表現:
  - 「買い下がる」
  - 「お婆ちゃんドヤ」
  - 「どんだけMなの？」
  - 「揺らぐジェンガ」
  - 「無双モード」
- 絵文字の戦略的使用

**減点対象**:
- 硬い文体、ビジネス文書のような表現
- 特徴的な表現が全くない
- 疑問形が少ない

### 4. 文章構造の一致度（20%）

**期待される構造**:
1. **導入**（疑問提起）
   - ニュースへの反応
   - 読者への問いかけ
2. **展開**（分析）
   - 「多分ね、こういうことなんです」等の説明
   - 二項対立の提示
   - 構造的な問題への深掘り
3. **結論**（示唆）
   - 投資家への実践的なアドバイス
   - 長期的な視点

**リズム**:
- 短文と長文の組み合わせ
- メイン60-70%、サブ30-40%のバランス

**減点対象**:
- 単調な文章構成
- 導入・展開・結論の流れがない
- リズムがない

### 5. 思考パターンの再現度（20%）

**期待される思考パターン**:
- 二項対立の提示
  - 例: 「表面的な見方 vs プロの見方」
  - 例: 「富裕層 vs 貧困層」
- 構造的な問題への深掘り
  - 例: 「格差の構造」「揺らぐジェンガ」
- ゴールドマン・サックス時代の経験活用
  - 例: 「ゴールドマンにいた頃」「投資銀行時代」
- 因果関係の明示
  - 例: 「なぜなら〜」「そもそも〜」

**減点対象**:
- 表面的な分析のみ
- ゴールドマン・サックスへの言及がない
- 因果関係が不明確

### 6. 価値観の織り込み度（10%）

**期待される価値観**:
- リベラルな視点
  - 例: 「フェア精神」「健全な証拠」
- 格差問題への言及
  - 例: 「株価が最高値でも、あなたの暮らし、良くなってる？」
  - 例: 「富裕層だけが恩恵を受けて、格差は広がる一方」
- 女性エンパワメント
  - 例: 「お婆ちゃんになった時に孫にドヤる」

**減点対象**:
- 価値観が全く織り込まれていない
- 中立的すぎる表現

## 評価の実施方法

### ステップ1: 各項目を個別に評価

1. 言語・方言（10点満点）
2. フォーマットの正確性（15点満点）
3. 口調・言い回しの再現度（25点満点）
4. 文章構造の一致度（20点満点）
5. 思考パターンの再現度（20点満点）
6. 価値観の織り込み度（10点満点）

### ステップ2: 合計点を計算

合計点 = 各項目の点数の合計

### ステップ3: 目標達成判定

- 90点以上: 合格、次のステップへ
- 90点未満: 不合格、改善して再評価
- 10回イテレーション達成: 強制的に次のステップへ

### ステップ4: 改善の実施

不合格の場合:
1. 正解データ（10本のサンプルコラム）と比較
2. 何が違うのかを具体的に列挙
3. なぜその差分が生まれたのかを分析
4. 分析結果に基づいて修正
5. 再評価

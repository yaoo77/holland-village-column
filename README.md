# Holland Village 経済コラム自動配信システム

河村真紀子様（ゴールドマン・サックス出身の金融専門家）のオンラインサロン「Holland Village」向けに、毎朝8時に経済ニュースコラムを自動配信するLINE botシステム。

## ファイル構成

### 1. `lessons_learned.md`
反省文ナレッジ。過去のミス（関西弁使用、フォーマット違反等）と原因分析、対策を記録。スケジュール実行時に必ず最初に参照する。

### 2. `evaluation_criteria_final.md`
コラム評価基準（言語10%、フォーマット15%、口調25%、文章構造20%、思考パターン20%、価値観10%）。目標90点以上。

### 3. `column_format_guide.md`
正しいコラムフォーマットガイド。タイトル、情報源、導入部、改行、文字数（700文字前後）、締めの形式を定義。

### 4. `column_templates.md`
10本のサンプルコラムから抽出した10個の型（型1〜10）。サンプルBで使用。

### 5. `ab_approach_guide.md`
サンプルAとBのアプローチの違い。
- サンプルA: 型を使わず、AIが自由に構成
- サンプルB: 型を選択して忠実に再現

## システム概要

### 実行フロー
1. 最新の経済ニュースをリサーチ（Bloomberg、日経等）
2. サンプルAを生成（型を使わない）
3. サンプルBを生成（型を選択）
4. 両方を評価（90点以上目標）
5. 高得点の方をLINE公式アカウントでブロードキャスト送信

### スケジュール
毎朝8時（日本時間）に自動実行

### 技術スタック
- LINE Messaging API（Channel ID: 2007065333）
- LINE MCPコネクタ
- GitHub（ナレッジ管理）
- Manus スケジュール機能

## 重要な注意事項

### 絶対に守ること
1. **標準語を使用する**（関西弁は絶対に使わない）
2. **フォーマットを厳守する**
3. **文字数700文字前後**（±100文字）
4. **絵文字を使用する**
5. **サンプルAでは型を使わない**
6. **サンプルBでは型を忠実に再現する**
7. **ニュースの事実を歪めない**
8. **評価基準で90点以上を目指す**
